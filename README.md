# Noir Fashion Week Website Redesign

A modern, premium redesign of the Noir Fashion Week website featuring world-class design patterns, comprehensive functionality, and an exceptional user experience.

## Project Overview

This repository contains the complete redesign planning and HTML prototype for the Noir Fashion Week website. The project includes competitive research, design system documentation, and a fully functional prototype ready for team review and collaboration.

## Quick Start

### View the Prototype

1. Navigate to the prototype directory:
   ```bash
   cd prototype
   ```

2. Open `index.html` in your web browser:
   ```bash
   open index.html
   ```
   Or simply double-click the file in Finder

3. Explore all pages using the navigation menu

## Key Features by Page

### Homepage
- Full-screen hero with call-to-actions
- Featured designers showcase
- Upcoming shows preview
- About section with mission
- Statistics and achievements
- Press coverage display
- Contact form

### Schedule
- Countdown to next event
- Category filtering (runway, presentation, special events)
- Day-by-day event breakdown
- Time, location, and details for each show
- Download full schedule option

### Designers
- Search by name or description
- Filter by category (womenswear, menswear, sustainable, etc.)

- Collection links

### Press
- Media statistics dashboard

- Press contact details

## Design Principles

1. **Elegance Through Simplicity** - Clean layouts, minimal color palette
2. **Premium Visual Hierarchy** - Bold typography, clear structure
3. **Content-First** - High-quality imagery takes center stage
4. **Sophisticated Interactions** - Smooth, refined animations
5. **Timeless Aesthetic** - Classic foundation with modern touches

## Technology Stack

### Current Prototype
- HTML5 (semantic markup)
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (no frameworks)
- Google Fonts (Playfair Display, Inter, JetBrains Mono)

### Recommended for Production
- **Frontend**: Next.js with React
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **CMS**: Sanity or Contentful
- **Hosting**: Vercel or Netlify
